# 数据类型映射

工具会自动识别源数据的类型，并在目标数据库中创建合适的字段类型：

| 源数据类型 | 目标数据类型 | 说明 |
|-----------|-------------|------|
| 整数（1, 2, 3...） | INTEGER | 存储整数 |
| 小数（3.14, 99.99） | REAL | 存储小数 |
| 文本（abc, 你好） | TEXT | 存储文字 |
| 日期（2024-01-01） | TEXT | 存储为文本格式（ISO 8601） |
| 二进制（图片、文件） | BLOB | 存储二进制数据 |

## 映射细节

- **整数类型**：映射为目标数据库的 `INTEGER`
- **小数/浮点类型**：映射为 `REAL`，支持精度感知
- **字符串类型**（VARCHAR、CHAR、TEXT 等）：映射为 `TEXT`
- **日期/时间类型**：以 ISO 8601 格式存储为 `TEXT`
- **二进制类型**（BLOB、RAW、BINARY 等）：映射为 `BLOB`

具体的目标类型会根据目标数据库引擎的不同而有所变化。db-ferry 使用每个适配器的类型映射函数将通用类型转换为引擎特定的 DDL。
